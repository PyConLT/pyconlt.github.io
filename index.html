PyCon Lithuania is annual conference that brings together hundreds of Python and data enthusiasts and professionals.

Get Tickets Become a speaker Become a partner

PyCon LT is a community event that brings together new and experienced Python users

PyCon LT is organized since 2009.

Our goals:

Grow and support a community of Python users
Encourage learning and knowledge sharing
Popularize Python tools/libraries and open source in general

Tracks

Python

PyCon Lithuania is about Python programming language, it's ecosystem, tooling, applications.

You can expect anything in this track. Talks about best practices, libraries, frameworks, solution; applications in Web Development, Devops, Security, Sience and more.

PyData is a community for developers and users of open source data tools. On meetup.com platform PyData is in 68 countries, has 198 groups and 191227 member.

You can expect talks about data analysis, science, ML engineering, ETL pipelines and so on. More

Education Summit

Newest addition to PyCon LT, Education summit welcomes everyone who are interested in using Python for education purposes. We promote use of Python in secondary and higher education settings.

All secondary education students and teacher are welcome to PyCon Lithuania for free!
